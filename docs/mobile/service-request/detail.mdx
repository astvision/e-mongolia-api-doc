## Үйлчилгээний хүсэлтийн дэлгэрэнгүй

### Сервис хаяг

```text
GET https://service.e-mongolia.mn/mobile-api/api/service-request/{serviceRequestId}
```

### Header параметрүүд

Header дээр **Authentication token** оруулах шаардлагатай юм.

```text
X-Auth-Token: // string шаардлагатай
```

### Path параметрүүд

```js
String serviceRequestId // хүсэлтийн дугаар
```
