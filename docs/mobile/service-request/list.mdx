## Үйлчилгээний хүсэлтийн жагсаалт авах

### Сервис хаяг

```text
GET https://service.e-mongolia.mn/mobile-api/api/service-request
```

### Header параметрүүд

Header дээр **Authentication token** оруулах шаардлагатай юм.

```text
X-Auth-Token: // string шаардлагатай
```

### Query параметрүүд

Дээрх сервис хаягыг доорх query параметрүүдтэй цуг дуудна. Бүх параметрүүдийг бичих шаардлагагүй. Шаардлагатай параметрүүдээ оруулахад болно.

```javascript
String status // Төлөв Enum:[SUBMITTED, PROCESSING, EDIT, FILE_REQUESTED, PRINTING, COMPLETED, PRINTED, REJECTED, EXPIRED, GRANTED]
String number // Хүсэлтийн дугаар
LocalDate createdDate // Үүсгэсэн огноо
int currentPage // Одоогийн хуудас (0-оос эхэлнэ)
int pageSize // Нэг хуудсанд харуулах тоо (default 10)
String sortDirection // Эрэмбэлэх чиглэл Enum: [ASC, DESC]
String[] sortParams // Эрэмбэлэх талбарууд
```
