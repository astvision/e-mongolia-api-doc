## Үйлчилгээний хүсэлтийн мэдээлэл үүсгэх

Төлбөртэй үйлчилгээ бол төлбөр төлөгдөх үед төлөв нь солигдоно, төлбөргүй үйлчилгээ бол /service-request/set-submitted-free API-г дуудаж төлөвийг солино.

### Сервис хаяг

```text
POST https://e-mongolia.mn/api/service-request/create
```

### Header параметрүүд

Header дээр **Authentication token** оруулах шаардлагатай

```javascript
X-Auth-Token: // string шаардлагатай
```

### Request body

```java
String serviceId // Үйлчилгээний ID
Object formData // 
String regnum // 
Object citizen // 
String xypRequestId // 
Object data // 
String representativeType // 
String representativeRegnum // 
String representativeReason // 
String representativeReasonFile // 
```
